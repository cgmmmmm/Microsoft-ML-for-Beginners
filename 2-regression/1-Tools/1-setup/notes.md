# Checklist of Items (Setup)

1. VS Code
2. Python version 3
3. Python Virtual Environment (.venv)
4. Essential Python Packages:
    * **pandas** (Data Analysis)
    * **matplotlib** (Data Visualization)
    * **numpy** (Scientific Computing)
    * **scikit-learn** (Predictive Analysis)
    * **ipykernel** (Jupyter notebook backend)

<br>

Run this command to install the essential python packages, inside your python virtual environment:
```
pip install pandas matplotlib numpy scikit-learn ipykernel
```

<br>

To verify package has been installed, run this command:
```
pip show pandas matplotlib numpy scikit-learn ipykernel
```
Your terminal should display detailed information about these python packages. The output is presented in a readable, RFC-compliant mail header format and includes metadata about the package.
